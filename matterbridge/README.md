# Home Assistant Add-on: Matterbridge

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

_A Matter plugin manager that allows you to have all your Matter devices up and running in minutes without dealing with complex pairing processes._

## About

This uses the latest official unmodified [Docker image of Matterbridge](https://hub.docker.com/r/luligu/matterbridge) as a Home Assistant Add-on.
